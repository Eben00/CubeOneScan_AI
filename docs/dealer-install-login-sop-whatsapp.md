# Dealer Install + Login SOP (WhatsApp Ready)

Use this quick guide to install the CubeOne Scan app and log in.

## 1) Install the App (Android)

1. Open your camera and scan the QR code shared by your manager.
2. Tap the link and download the APK file.
3. Open your `Files` app, then go to `Downloads`.
4. Tap the downloaded APK (usually starts with `app-evolvesa-arm64-v8a`).
5. If Android asks for permission, allow install from this source.
6. Tap `Install`, then `Open`.

## 2) Samsung / Android Security Warning Path

If you see `Blocked` or `For your security, your phone is not allowed to install unknown apps`:

1. Tap `Settings` on that pop-up.
2. Enable `Allow from this source` (or `Install unknown apps` for `My Files`/browser).
3. Go back to `Downloads` in `Files`.
4. Tap the APK again and complete install.

## 3) Single Stable Download Flow

- Use the direct arm64 APK link/QR only (official pilot link from CubeOne team).
- Do not use old APK links from previous messages.

## 4) If the Install Pop-Up Loops

1. Close installer and browser completely.
2. Open `Files` > `Downloads`.
3. Delete older app APKs and keep only the newest one.
4. Tap the newest APK again.
5. If still looping, restart phone once and retry from `Files`.

## 5) First Login

1. Open the app.
2. Enter the dealer login email and password provided by manager.
3. Wait for home screen to load.
4. If login fails, confirm mobile data/Wi-Fi, then retry once.
